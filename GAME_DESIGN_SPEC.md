# 땅콩전사 키우기 — 게임 설계 기준서

이 문서는 초기 개발 요구사항과 이후 확정된 변경사항을 하나로 정리한 **현재 최상위 기준 문서**다. 구현과 테스트가 이 문서와 충돌하면 본 문서를 우선한다.

## 1. 게임 정체성

- 장르: 가로형 모바일 자동 방치 RPG
- 제목: `땅콩전사 키우기`
- 핵심 플레이: 자동 사냥 → 성장 → 보스 자동 도전 → 다음 스테이지
- 조작 방향: 장시간 켜 두어도 진행되는 완전 자동 전투
- 보스전: 수동 회피나 패턴 대응 없이 전투력·성장·제한시간으로 승패 결정
- 시각 방향: 밝고 귀여운 판타지 땅콩 세계
- 타사 게임은 정보 구조와 사용성만 참고하며 명칭·캐릭터·아이콘·화면을 복제하지 않는다.

## 2. 확정 메뉴

### 하단 고정 메뉴 순서

1. `성장`
2. `장비`
3. `스킬`
4. `펫`
5. `전직`
6. `상점`

### 별도 접근

- 스테이지 선택: 상단 스테이지 제목 옆 `선택` 버튼
- 설정: 우측 상단 설정 버튼
- 모험·임무·빠른 메뉴·껍질 단련·속성 연구·방치 효율 메뉴는 사용하지 않는다.

## 3. 스테이지 진행

- 스테이지는 무한히 이어진다.
- 한 월드는 30개 스테이지로 구성한다.
- 일반 몬스터 100마리를 처치하면 보스 도전 조건을 달성한다.
- 보스 자동 도전 ON: 100마리 달성 즉시 보스전 진입
- 보스 자동 도전 OFF: 100/100을 유지한 채 일반 사냥과 재화 획득을 계속한다.
- 보스 입장 시 HP·MP·기본 공격 쿨타임·모든 스킬 쿨타임을 초기화한다.
- 보스 처치 시 다음 스테이지로 이동한다.
- 보스전 실패 시 현재 스테이지 0/100부터 다시 사냥한다.
- 일반 사냥 중 사망 시 이전 스테이지 0/100부터 다시 시작한다.
- 보스전 중 다른 스테이지로 이동할 수 없다.
- 최초 보스 클리어 보상은 스테이지별 한 번만 지급한다.

## 4. 보스전

- 수동 회피 패턴을 사용하지 않는다.
- 보스전 제한시간은 45초를 기본으로 한다.
- 난이도는 보스 HP·공격력·스테이지 배율·제한시간으로 조절한다.
- 자동 공격과 전체 스킬 AUTO 상태를 그대로 사용한다.
- 보스 이름과 외형은 월드별로 다르게 구성한다.

## 5. 성장

성장 화면은 좌우로 나눈다.

- 왼쪽: 현재 땅콩 외형, 전직 명칭, 레벨, 전투력, 공격력, HP, MP, 치명타, 장착 장비
- 오른쪽: 능력치 강화 항목과 ×1·×10·×100 강화 버튼

성장 항목은 아래 10개만 존재한다.

1. 공격력
2. HP
3. HP 회복량
4. MP
5. MP 회복량
6. 치명타
7. 치명타 피해량
8. 경험치 획득량 증가
9. 골드 획득량 증가
10. 장비 강화 획득량 증가

규칙:

- 치명타 확률은 100%가 최대다.
- 100% 달성 후 추가 골드를 소모하지 않는다.
- 경험치는 일반 몬스터와 보스 처치로 획득한다.
- 골드 획득량 증가는 실제 사냥 보상에 적용한다.
- 장비 강화 획득량 증가는 실제 장비 강화 재료 획득에 적용한다.

## 6. 스킬

- 화면 왼쪽은 사냥 스킬 4개, 오른쪽은 보스 스킬 4개로 구분한다.
- 스킬은 박스형 카드 없이 원형 문양으로 배치한다.
- 원형 스킬에는 명칭·레벨·현재 쿨타임 상태를 표시한다.
- 원형 스킬을 누르면 스킬 설명창이 열린다.
- 설명창에는 간단 설명, 현재 레벨, 실제 총 피해량, 타격 수, 타격당 피해, 기본 쿨타임, MP 소모, 강화 비용을 표시한다.
- 스킬 강화는 설명창 안의 `스킬 강화` 버튼으로만 실행한다.
- 설명창의 피해량은 실제 전투 공식과 같은 값을 사용한다.
- 전체 AUTO 버튼은 화면 맨 왼쪽 상단에 작은 크기로 배치한다.
- 개별 AUTO는 사용하지 않는다.

## 7. 장비

장비 화면에는 사냥용·보스용 구분 탭을 두지 않는다.

### 속성 탭

1. `무속성`
2. `화염`
3. `냉기`
4. `번개`

장비 규칙:

- 장비는 4속성 × 4등급 × 3종으로 총 48개다.
- 등급은 `레어`, `에픽`, `유니크`, `레전드` 순서다.
- 장비 하나는 사냥 모드와 보스 모드에서 **공격 방식 자체가 다르게 작동**한다.
- 사냥 모드에서는 다중 타깃·범위 공격·연쇄 공격으로 여러 일반 몬스터를 동시에 공격한다.
- 보스 모드에서는 범위 공격을 중단하고 보스 한 명에게 모든 추가 피해를 집중한다.
- 장비 종류별 기본 전환 패턴은 아래와 같다.
  - 1번 장비: 사냥 `부채꼴 다중 베기` / 보스 `집중 참격`
  - 2번 장비: 사냥 `원형 범위 폭발` / 보스 `단일 연속 타격`
  - 3번 장비: 사냥 `연쇄 검격` / 보스 `처형 참격`
- 처형 참격은 보스 HP가 일정 비율 이하일 때 추가 피해를 준다.
- 장비 카드에는 사냥 공격 방식·최대 대상 수·범위·피해량과 보스 공격 방식·타수·집중 피해량을 각각 표시한다.
- 같은 장비를 사냥 슬롯과 보스 슬롯에 각각 장착할 수 있다.
- `사냥 장착`과 `보스 장착` 버튼은 장비 카드마다 별도로 둔다.
- 각 장비는 개별 보유 수량과 강화 레벨을 가진다.
- 중복 장비 3회 누적마다 해당 장비 레벨이 추가 상승한다.
- 장비 강화는 장비 강화 재료를 사용한다.
- 등급·레벨·중복·속성에 따라 대상 수, 범위, 타수와 피해량이 함께 증가한다.
- 기존 사냥용 48개와 보스용 48개 저장 데이터는 같은 장비끼리 합쳐 통합 장비 48개로 이전한다.

## 8. 전직

전직 명칭은 모두 `○○ 땅콩` 형식으로 사용한다.

1. 새싹 땅콩
2. 전투 땅콩
3. 황금 수호 땅콩
4. 화염 갑각 땅콩
5. 빙결 갑각 땅콩
6. 뇌광 갑각 땅콩
7. 왕실 갑주 땅콩
8. 차원 수호 땅콩

규칙:

- 전직 화면은 한 단계당 한 줄인 리스트 형식으로 구성한다.
- 모든 전직 항목에 `전직하기` 버튼 영역을 둔다.
- 완료 단계는 `전직 완료`, 현재 단계는 `현재 전직`, 이후 단계는 `잠김`으로 표시한다.
- 바로 다음 단계만 조건 충족 시 `전직하기` 버튼이 활성화된다.
- 전직 조건은 스테이지·전투력·골드·다이아를 사용한다.
- 전직 성공 시 능력치 배율과 기본 공격 타수가 증가한다.
- 전직 시 HP와 MP를 완전히 회복한다.
- 2차 전직 달성 시 펫 슬롯 3개를 동시에 해금한다.

## 9. 펫

- 펫 슬롯은 총 3개다.
- 2차 전직에서 3슬롯을 동시에 해금한다.
- 펫은 주인공과 독립적으로 이동하고 자동 공격한다.
- 펫은 가능한 한 서로 다른 몬스터를 맡으며 최소 간격을 유지한다.
- 보스전에서는 보스를 서로 다른 방향에서 포위한다.
- 펫은 HP와 MP가 없고 적의 공격 대상이 되지 않는다.
- 기본 속성은 화염·냉기·번개다.
- 사냥·보스 모두 자동 편성으로 동작한다.
- 알 구매와 부화를 통해 펫 성장 재화를 얻는다.
- 펫 성장 항목은 공격력·치명타 확률·치명타 피해량이다.

## 10. 방치 보상

- 앱을 종료한 시간 동안 골드·경험치·장비 강화 재료를 계산한다.
- 최고 해금 스테이지와 현재 전투력을 기준으로 효율을 계산한다.
- 최대 누적 시간은 8시간을 기본으로 한다.
- 동일 종료 시각을 기준으로 중복 보상을 지급하지 않는다.
- 방치 보상은 별도 성장 메뉴가 아닌 기본 시스템으로 제공한다.

## 11. 저장

- 자동 저장 주기: 10초
- 백업 저장 주기: 30초
- 앱 백그라운드 진입과 종료 시 즉시 저장
- 저장 데이터에 버전 번호를 둔다.
- 음수 재화와 잘못된 스테이지 범위를 자동 보정한다.
- 기존 PlayerPrefs 데이터는 정식 저장 구조로 마이그레이션한다.
- 기존 분리 장비 데이터는 통합 장비 48개로 자동 이전한다.
- 손상된 주 저장 파일이 있으면 백업 파일 복구를 시도한다.

## 12. 상점

- 오늘의 접속 보상은 하루 한 번만 지급한다.
- 검 장비 소환은 하나의 상품으로 제공한다.
- 검 소환은 다이아 5개를 사용한다.
- 소환한 장비는 사냥 패턴과 보스 패턴이 동시에 해금된다.
- 펫 알 구매를 제공한다.
- 실제 결제·광고는 운영 정책 확정 전에는 구현하지 않는다.

## 13. 설정

- 60 FPS / 30 FPS 절전 모드
- BGM 음량
- 효과음 음량
- 진동 ON/OFF
- 자동 저장 상태 표시
- 즉시 저장
- 설정은 우측 상단 메뉴에서만 접근한다.

## 14. 개발 완료 기준

- 하단 메뉴가 성장 → 장비 → 스킬 → 펫 → 전직 → 상점 순서로 표시된다.
- 성장 화면 왼쪽에 외형·스펙, 오른쪽에 강화 항목이 표시된다.
- 사냥·보스 스킬이 좌우로 분리되고 원형으로 표시된다.
- 스킬 문양을 누르면 실제 피해량이 포함된 설명창이 열린다.
- 장비 화면에는 속성 탭만 표시되고 사냥용·보스용 구분 탭은 없다.
- 사냥에서는 장비가 여러 몬스터를 범위 공격하고, 보스전에서는 한 명에게 집중 공격한다.
- 통합 장비 48개가 모드별 공격 방식과 수치를 함께 표시한다.
- 전직 8개가 리스트로 표시되고 각 항목 버튼 상태가 조건에 따라 달라진다.
- 펫 3마리가 산개하여 자동 전투하며 부화·성장·저장된다.
- 무한 스테이지·30단위 월드·100마리·보스 자동 도전 규칙이 작동한다.
- 설정과 저장이 재실행 후 유지된다.
- Unity Play Mode 자동 진단에서 차단 오류가 없어야 한다.
