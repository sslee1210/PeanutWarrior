# 땅콩전사 키우기 — 게임 설계 기준서

이 문서는 초기 개발 요구사항과 이후 확정된 변경사항을 하나로 정리한 **현재 최상위 기준 문서**다. 구현과 테스트가 이 문서와 충돌하면 본 문서를 우선한다.

## 1. 게임 정체성

- 장르: 가로형 모바일 자동 방치 RPG
- 제목: `땅콩전사 키우기`
- 핵심 플레이: 자동 사냥 → 성장 → 보스 자동 도전 → 다음 스테이지
- 조작 방향: 장시간 켜 두어도 진행되는 완전 자동 전투
- 보스전: 수동 회피나 패턴 대응 없이 전투력·성장·제한시간으로 승패 결정
- 시각 방향: 밝고 귀여운 판타지 땅콩 세계
- 타사 게임은 정보 구조와 사용성만 참고하며 명칭·캐릭터·아이콘·화면을 복제하지 않는다.

## 2. 확정 메뉴

### 하단 고정 메뉴

1. `SKILL`
2. `장비`
3. `성장`
4. `전직`
5. `펫`
6. `상점`

### 별도 접근

- 스테이지 선택: 상단 스테이지 제목 옆 `선택` 버튼
- 설정: 우측 상단 설정 버튼
- 모험·임무·빠른 메뉴·껍질 단련·속성 연구·방치 효율 메뉴는 사용하지 않는다.

## 3. 스테이지 진행

- 스테이지는 무한히 이어진다.
- 한 월드는 30개 스테이지로 구성한다.
- 일반 몬스터 100마리를 처치하면 보스 도전 조건을 달성한다.
- 보스 자동 도전 ON: 100마리 달성 즉시 보스전 진입
- 보스 자동 도전 OFF: 100/100을 유지한 채 일반 사냥과 재화 획득을 계속한다.
- 보스 입장 시 HP·MP·기본 공격 쿨타임·모든 스킬 쿨타임을 초기화한다.
- 보스 처치 시 다음 스테이지로 이동한다.
- 보스전 실패 시 현재 스테이지 0/100부터 다시 사냥한다.
- 일반 사냥 중 사망 시 이전 스테이지 0/100부터 다시 시작한다.
- 보스전 중 다른 스테이지로 이동할 수 없다.
- 최초 보스 클리어 보상은 스테이지별 한 번만 지급한다.

## 4. 보스전

- 수동 회피 패턴을 사용하지 않는다.
- 보스전 제한시간은 45초를 기본으로 한다.
- 난이도는 보스 HP·공격력·스테이지 배율·제한시간으로 조절한다.
- 자동 공격과 전체 SKILL AUTO 상태를 그대로 사용한다.
- 보스 이름과 외형은 월드별로 다르게 구성한다.

## 5. 성장

성장 화면에는 아래 10개 항목만 존재한다.

1. 공격력
2. HP
3. HP 회복량
4. MP
5. MP 회복량
6. 치명타
7. 치명타 피해량
8. 경험치 획득량 증가
9. 골드 획득량 증가
10. 장비 강화 획득량 증가

규칙:

- 강화 수량은 ×1, ×10, ×100을 지원한다.
- 치명타 확률은 100%가 최대다.
- 100% 달성 후 추가 골드를 소모하지 않는다.
- 경험치는 일반 몬스터와 보스 처치로 획득한다.
- 골드 획득량 증가는 실제 사냥 보상에 적용한다.
- 장비 강화 획득량 증가는 실제 장비 강화 재료 획득에 적용한다.
- 별도 껍질 단련·속성 연구·방치 효율 성장 항목은 만들지 않는다.

## 6. 전직

- 전직은 독립 메뉴로 제공한다.
- 전직 조건은 스테이지·전투력·골드·다이아를 사용한다.
- 전직 성공 시 능력치 배율과 기본 공격 타수가 증가한다.
- 전직 시 HP와 MP를 완전히 회복한다.
- 2차 전직 달성 시 펫 슬롯 3개를 동시에 해금한다.
- 전직 외형은 땅콩 껍질의 성장과 수호자 진화를 중심으로 구성한다.

## 7. 펫

- 펫 슬롯은 총 3개다.
- 2차 전직에서 3슬롯을 동시에 해금한다.
- 펫은 주인공과 독립적으로 이동하고 자동 공격한다.
- 펫은 HP와 MP가 없고 적의 공격 대상이 되지 않는다.
- 기본 속성은 화염·냉기·번개다.
- 사냥·보스 모두 자동 편성으로 동작한다.
- 알 구매와 부화를 통해 펫 성장 재화를 얻는다.
- 펫 성장 항목은 공격력·치명타 확률·치명타 피해량이다.

## 8. 방치 보상

- 앱을 종료한 시간 동안 골드·경험치·장비 강화 재료를 계산한다.
- 최고 해금 스테이지와 현재 전투력을 기준으로 효율을 계산한다.
- 최대 누적 시간은 8시간을 기본으로 한다.
- 동일 종료 시각을 기준으로 중복 보상을 지급하지 않는다.
- 방치 보상은 별도 성장 메뉴가 아닌 기본 시스템으로 제공한다.

## 9. 저장

- 자동 저장 주기: 10초
- 백업 저장 주기: 30초
- 앱 백그라운드 진입과 종료 시 즉시 저장
- 저장 데이터에 버전 번호를 둔다.
- 음수 재화와 잘못된 스테이지 범위를 자동 보정한다.
- 기존 PlayerPrefs 데이터는 정식 저장 구조로 마이그레이션한다.
- 손상된 주 저장 파일이 있으면 백업 파일 복구를 시도한다.

## 10. 상점

- 오늘의 접속 보상은 하루 한 번만 지급한다.
- 펫 알 구매를 제공한다.
- 장비·SKILL 관련 상품과 확률은 해당 시스템 재설계 전까지 확장하지 않는다.
- 실제 결제·광고는 운영 정책 확정 전에는 구현하지 않는다.

## 11. 설정

- 60 FPS / 30 FPS 절전 모드
- BGM 음량
- 효과음 음량
- 진동 ON/OFF
- 자동 저장 상태 표시
- 즉시 저장
- 설정은 우측 상단 메뉴에서만 접근한다.

## 12. 현재 설계 보류 영역

아래 두 영역은 사용자가 별도로 확정할 때까지 현재 데이터 유지와 기본 화면만 제공한다.

### SKILL

- 스킬 종류
- 장착 슬롯 수
- 사냥·보스 프리셋
- 등급·획득·진화 방식
- 세부 밸런스

확정 사항: 개별 AUTO는 사용하지 않고 하나의 `전체 SKILL AUTO` 버튼만 사용한다.

### 장비/검

- 상세 장비 종류와 부위
- 옵션
- 강화·합성 방식
- 소환 확률과 천장
- 도감 효과

확정 사항:

- 무속성·화염·냉기·번개
- 등급 순서 `레어 → 에픽 → 유니크 → 레전드`
- 사냥 장착과 보스 장착 분리

## 13. 개발 완료 기준

스킬과 장비를 제외한 코드 기능 완료 기준:

- 성장 10종이 실제 전투·보상에 적용된다.
- 전직 조건·효과·저장·펫 해금이 연결된다.
- 펫 3마리가 표시되고 자동 전투하며 부화·성장·저장된다.
- 무한 스테이지·30단위 월드·100마리·보스 자동 도전 규칙이 작동한다.
- 최초 클리어·일일 보상·방치 보상이 중복 없이 지급된다.
- 설정과 저장이 재실행 후 유지된다.
- 모바일 Canvas의 메뉴가 겹치지 않고 안전영역에 맞는다.
- Unity Play Mode 자동 진단에서 차단 오류가 없어야 한다.
