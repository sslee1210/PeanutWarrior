# 땅콩전사 키우기 — 메뉴 레이아웃 V2 검증표

## 공통 원칙

- 하단 메뉴는 `SKILL · 장비 · 성장 · 전직 · 펫 · 상점` 6개만 표시한다.
- 선택 메뉴는 배경색을 진하게 바꾸지 않고 하단 금색 선으로 구분한다.
- 버튼은 마우스 오버와 선택 상태에서 배경색이 갑자기 진해지지 않는다.
- 각 화면은 `상단 요약 → 현재 상태 → 주요 행동` 순서로 읽힌다.
- 설정은 우측 상단 설정 버튼에서만 접근한다.
- 스테이지 선택은 메인 화면의 스테이지 제목 옆 `선택` 버튼에서 접근한다.

## SKILL

- 상단에 전체 `AUTO ON/OFF` 버튼이 하나만 표시된다.
- 개별 스킬 AUTO 버튼은 표시되지 않는다.
- 사냥 스킬 4개와 보스 스킬 4개가 2열 카드로 표시된다.
- 각 카드에서 이름, 레벨, 쿨타임, 강화 비용을 바로 확인할 수 있다.
- 스킬 구성은 재설계 전 상태임을 표시한다.

## 장비

- 상단에서 현재 사냥 장비와 보스 장비를 동시에 확인할 수 있다.
- 무속성·화염·냉기·번개 장비가 2×2 카드로 표시된다.
- 각 카드에서 최고 등급, 레벨, 등급별 보유 수량을 확인할 수 있다.
- 등급은 `레어 → 에픽 → 유니크 → 레전드` 순서만 사용한다.
- 사냥 장착과 보스 장착 버튼을 분리한다.
- 세부 옵션·강화·합성 구성은 재설계 전 상태로 유지한다.

## 성장

- 성장 항목 10개가 2열 5행으로 한 화면에 표시된다.
- 공격력, HP, HP 회복량, MP, MP 회복량, 치명타, 치명타 피해량, 경험치 획득량 증가, 골드 획득량 증가, 장비 강화 획득량 증가만 존재한다.
- 강화 수량은 ×1, ×10, ×100으로 선택한다.
- 치명타는 100%에서 MAX가 되며 추가 골드를 소모하지 않는다.
- 각 카드에서 현재 수치, 레벨, 강화 수량, 골드 비용을 확인할 수 있다.

## 전직

- 현재 전직 외형·단계·효과를 왼쪽에서 확인한다.
- 전체 전직 단계를 중앙 진행 카드에서 확인한다.
- 다음 전직의 스테이지·전투력·골드·다이아 조건을 오른쪽에서 확인한다.
- 모든 조건 충족 전에는 전직 버튼이 비활성화된다.
- 2차 전직 시 펫 3슬롯 해금 문구가 표시된다.

## 펫

- 상단에 펫 슬롯 3개와 해금 상태를 표시한다.
- 화염·냉기·번개 펫 슬롯을 구분한다.
- 알 보유량, 부화 도감, 부화 남은 시간을 확인한다.
- 펫 공격력·치명타 확률·치명타 피해 강화를 한 영역에서 확인한다.
- 알 구매와 부화 시작 버튼을 분리한다.

## 상점

- 상단에서 최근 상점 메시지와 보유 재화를 확인한다.
- 오늘의 보급, 사냥 장비 소환, 보스 장비 소환, 펫 알을 2×2 카드로 표시한다.
- 각 카드에 설명, 가격, 실행 버튼을 함께 표시한다.

## 스테이지 선택

- 이전·다음 월드 버튼과 최고 해금 스테이지를 상단에 표시한다.
- 한 월드의 30개 스테이지를 6×5 격자로 표시한다.
- 현재 스테이지는 금색 테두리와 `현재` 문구로 구분한다.
- 미해금 스테이지는 비활성화한다.
- 보스전 중에는 스테이지 이동을 차단한다.

## 설정

- 60 FPS, 30 FPS 절전, 즉시 저장, 현재 환경을 2×2 카드로 표시한다.
- 즉시 저장은 핵심 저장, 성장, 펫 데이터를 함께 저장한다.

## 자동 검사

Play Mode에서 아래 로그를 확인한다.

```text
[PeanutWarrior Menu Audit]
PASS · eight redesigned inner pages, two-column growth cards, fixed button colors and underline navigation are active.
```

Unity 에디터에서 `Peanut Warrior > Validation > Run EditMode Tests`를 실행하면 V2 메뉴 계약 테스트도 함께 실행된다.
