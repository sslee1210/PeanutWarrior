# 땅콩전사 키우기 — 구현 상태 및 검증 순서

게임 규칙의 최상위 기준은 저장소 루트의 [`GAME_DESIGN_SPEC.md`](GAME_DESIGN_SPEC.md)다.

## 1. 현재 구현 완료 범위

### 하단 메뉴

- 성장
- 장비
- 스킬
- 펫
- 전직
- 상점

표시 순서, 클릭 대상, 선택 강조가 같은 순서로 유지된다.

### 성장

- 왼쪽: 땅콩 외형, 현재 전직, 레벨, 전투력, 공격력, HP, MP, 치명타, 사냥용·보스용 장비
- 오른쪽: 성장 능력치 10개
- ×1, ×10, ×100 강화
- 치명타 100% MAX 제한
- 경험치·골드·장비 강화 재료 획득 증가 실제 적용

### 스킬

- 박스형 카드 제거
- 사냥 스킬 4개를 왼쪽에 배치
- 보스 스킬 4개를 오른쪽에 배치
- 스킬을 원형 버튼으로 표시
- 원형 버튼에 이름·레벨·쿨타임 표시
- 원형 버튼 클릭 시 조각으로 강화
- 전체 AUTO 버튼을 화면 왼쪽 상단에 작은 크기로 배치

### 장비

장비는 사냥용과 보스용이 서로 다른 도감이다.

#### 사냥용 검

- 무속성·화염·냉기·번개 4속성
- 레어·에픽·유니크·레전드 4등급
- 속성·등급별 3개
- 총 48개

#### 보스용 검

- 무속성·화염·냉기·번개 4속성
- 레어·에픽·유니크·레전드 4등급
- 속성·등급별 3개
- 총 48개

#### 공통 기능

- 전체 장비 총 96개
- 상단 `사냥용 검 / 보스용 검` 탭
- 각 장비군 내부에 4속성 탭
- 개별 보유 수량과 강화 레벨
- 중복 3회마다 레벨 증가
- 장비 강화 재료 사용
- 사냥용 장비는 사냥 슬롯에만 장착
- 보스용 장비는 보스 슬롯에만 장착
- 현재 전투 상태에 맞는 장비 피해 보너스 적용
- 기존 48개 공유 장비 저장은 양쪽 장비군으로 마이그레이션

### 상점

- 사냥용 검 소환과 보스용 검 소환을 별도 상품으로 제공
- 각 소환 비용 다이아 5개
- 소환 결과는 선택한 용도의 장비 도감에만 등록
- 오늘의 접속 보상
- 성장 골드 보급
- 펫 알 구매
- 부화 즉시 완료

### 전직

전직 명칭은 모두 `○○ 땅콩` 형식이다.

1. 새싹 땅콩
2. 전투 땅콩
3. 황금 수호 땅콩
4. 화염 갑각 땅콩
5. 빙결 갑각 땅콩
6. 뇌광 갑각 땅콩
7. 왕실 갑주 땅콩
8. 차원 수호 땅콩

- 한 단계당 한 줄인 리스트 형식
- 각 단계에 개별 버튼
- 완료 단계: `전직 완료`
- 현재 단계: `현재 전직`
- 바로 다음 단계: 조건 충족 시 `전직하기` 활성화
- 이후 단계: `잠김`
- 스테이지·전투력·골드·다이아 조건 적용

### 펫

- 불씨 땅콩·서리 땅콩·번개 땅콩
- 3슬롯
- 독립 이동·자동 공격
- 무적·공격 대상 제외
- 레벨·별 승급·중복 조각

### 진행·보스·방치·저장

- 무한 스테이지
- 월드당 30스테이지
- 100마리 처치 후 보스 도전
- 보스 자동 도전
- 45초 제한시간
- 최대 8시간 방치 보상
- JSON 저장 스키마 v3
- 10초 자동 저장·30초 백업

## 2. Unity 확인 순서

1. Unity를 완전히 종료한다.
2. `git pull`을 실행한다.
3. Unity 6.3 LTS로 프로젝트를 연다.
4. Console에 빨간 오류가 있으면 가장 위의 첫 번째 오류를 확인한다.
5. 컴파일 완료 후 Play를 실행한다.
6. 하단 메뉴가 `성장 → 장비 → 스킬 → 펫 → 전직 → 상점` 순서인지 확인한다.
7. 성장 화면이 좌우로 나뉘는지 확인한다.
8. 스킬이 원형으로 표시되고 사냥·보스가 좌우로 나뉘는지 확인한다.
9. 장비 화면 상단에 `사냥용 검 / 보스용 검` 탭이 있는지 확인한다.
10. 각 용도 탭 안에 무속성·화염·냉기·번개 탭이 있는지 확인한다.
11. 각 속성에 등급별 장비 3개가 표시되는지 확인한다.
12. 사냥용 검을 보스 슬롯에 장착할 수 없는지 확인한다.
13. 상점에서 사냥용·보스용 검 소환이 각각 동작하는지 확인한다.
14. 전직 8개가 리스트로 표시되고 버튼 상태가 달라지는지 확인한다.

정상이라면 Console에 다음 로그가 표시된다.

```text
[PeanutWarrior Core Completion Audit]
PASS · navigation order, split growth, circular hunting/boss skills, per-tier advancement, separate 48-item hunting and 48-item boss sword catalogs, equipment V5 and separate summons are active.
```
