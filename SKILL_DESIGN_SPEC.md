# 땅콩전사 키우기 — 확정 스킬 설계

이 문서는 사냥 스킬 4개와 보스 스킬 4개의 현재 확정 구성, AUTO 사용 규칙, 전직 강화 규칙을 정의한다.

## 사냥 스킬

### 1. 껍질 회전참

황금 갑각검을 검륜처럼 펼치고 적 무리 사이를 질주한다. 이동 경로에 원형 검흔을 남기고 마지막에 검흔을 시간차로 폭발시킨다.

- 역할: 이동형 광역 공격
- 기본 MP: 20
- 기본 쿨타임: 6초
- 기본 타격 수: 6타
- 기본 대상 수: 최대 6명
- 기본 피해 배율: 공격력의 280%
- 전용 연출: 황금 갑각검 8개, 회전 검륜, 이동 지점 검흔 폭발
- 전직 진화: 전직마다 돌진 타격 +1, 대상 +1, 갑각검 연출 +2

### 2. 낙화검우

하늘에 거대한 황금 땅콩꽃이 피어나고 꽃잎이 추적 검날로 변해 전장에 쏟아진다. 땅에 박힌 검들은 황금선으로 연결된 뒤 동시에 폭발한다.

- 역할: 전장 추적 검비·연결 폭발
- 기본 MP: 25
- 기본 쿨타임: 9초
- 기본 타격 수: 12타
- 기본 대상 수: 최대 8명
- 기본 피해 배율: 공격력의 460%
- 전용 연출: 황금꽃 개화, 추적 검비 20개, 전장 연결 폭발
- 전직 진화: 전직마다 검비 +3타, 대상 +2, 파동 +1, 연출 검 +6

### 3. 지맥꼬투리진

검을 지면에 꽂아 황금 지맥으로 적들을 연결한다. 지맥을 따라 거대한 꼬투리 칼날이 연속으로 솟아오르고 마지막 대검이 적들을 띄운다.

- 역할: 지맥 설치·적 집결·공중 제어
- 기본 MP: 30
- 기본 쿨타임: 12초
- 기본 타격 수: 7타
- 기본 대상 수: 최대 8명
- 기본 피해 배율: 공격력의 540%
- 전용 연출: 지맥 방사선, 꼬투리 6개 분출, 마지막 대검 솟구침
- 전직 진화: 전직마다 분출 타격 +1, 대상 +2, 꼬투리 연출 +2, 끌어당김과 빙결 시간 강화

### 4. 왕실 꼬투리 천개

왕관 모양의 초대형 황금 꼬투리 무기고를 열어 수십 자루의 왕실 검을 전개한다. 검들이 전장을 절단한 뒤 하나의 거대검으로 합쳐져 지면을 내려친다.

- 역할: 화면 전체 사냥 필살기
- 기본 MP: 42
- 기본 쿨타임: 18초
- 기본 타격 수: 16타
- 기본 대상 수: 최대 12명
- 기본 피해 배율: 공격력의 880%
- 전용 연출: 왕실 꼬투리 개방, 왕실 검 24개 전개, 거대검 낙하
- 전직 진화: 전직마다 왕실 검 타격 +4, 대상 +3, 연출 검 +8, 고전직에서 거대검 마무리 수 증가

## 보스 스킬

### 5. 갑각해방

껍질을 여러 개의 갑각검 날개로 해방한다. 갑각검은 보스를 다른 각도에서 찌르고, 해방 시간 동안 기본 공격 속도를 높인 뒤 검진으로 붕괴한다.

- 역할: 공격 형태 해방·공격속도 강화
- 기본 MP: 22
- 기본 쿨타임: 10초
- 기본 타격 수: 6타
- 기본 피해 배율: 공격력의 360%
- 기본 강화 지속시간: 6초
- 전용 연출: 갑각검 날개 6개, 검진 수렴
- 전직 진화: 전직마다 갑각검 타격 +1, 갑각검 연출 +2, 지속시간 +0.75초, 공격속도 강화량 증가

### 6. 땅콩 연환검

보스 주위를 여러 방향으로 순간 이동하며 검흔을 남긴다. 마지막에 검을 거두면 모든 검흔이 동시에 폭발한다.

- 역할: 보스 단일 연속 공격
- 기본 MP: 30
- 기본 쿨타임: 13초
- 기본 타격 수: 9타
- 기본 피해 배율: 공격력의 720%
- 전직 진화: 전직마다 실제 연속 타격 +2, 잔상 검 +2, 고전직에서 검흔 폭발 파동 추가

### 7. 낙화귀근

보스 머리 위에 황금꽃을, 발밑에 뿌리 검진을 만든다. 두 검진이 전투 중 받은 피해를 저장하고 위아래에서 동시에 닫히며 축적 피해를 폭발시킨다.

- 역할: 피해 저장·상하 압축 폭발
- 기본 MP: 38
- 기본 쿨타임: 17초
- 기본 타격 수: 8타
- 기본 피해 배율: 공격력의 680%
- 기본 피해 저장률: 32%
- 전용 연출: 천상 황금꽃, 지하 뿌리 왕관, 상하 검날 수렴
- 전직 진화: 전직마다 압축 타격 +2, 연출 검 +3, 저장률 +4%, 유지시간 +0.25초

### 8. 황금핵 천단

땅콩전사의 생명핵을 태양처럼 빛나는 황금핵으로 만들고 모든 힘을 거대한 천상검에 집중한다. 내려칠 때마다 하늘과 지면을 함께 가른다.

- 역할: 보스 단일 초필살기
- 기본 MP: 55
- 기본 쿨타임: 24초
- 기본 타격 수: 1타
- 기본 피해 배율: 공격력의 1,450%
- 즉사 효과 없음
- 전용 연출: 황금핵 집속, 주변 검술 문양 흡수, 천상 거대검 낙하, 전장 충격파
- 전직 진화: 3단계부터 2단 천단, 6단계부터 3단 천단으로 변화하며 집속 검과 충격파 수가 계속 증가

## AUTO 발동 규칙

- 고정 스킬 우선순위를 사용하지 않는다.
- 각 스킬은 자신의 쿨타임이 끝나는 즉시 AUTO 대기열에 들어간다.
- 먼저 쿨타임이 끝난 스킬을 먼저 사용한다.
- 같은 프레임에 여러 스킬의 쿨타임이 끝나면 MP가 허용하는 범위에서 모두 연속 발동한다.
- 전투 시작에 해당 전투용 스킬 4개가 전부 READY라면 `개전 연속기`로 네 스킬을 같은 전투 틱에 겹쳐 사용한다.
- 개전 연속기는 네 스킬을 모두 발동한 뒤 보유 MP에서 전체 비용을 차감하며, 비용이 최대 MP보다 높으면 MP는 0이 된다.
- 이후에는 전직으로 감소된 실제 MP 비용과 실제 쿨타임을 사용한다.
- AUTO OFF 상태에서는 쿨타임이 끝나도 자동 발동하지 않는다.

## 전직 스킬 강화

전직 단계는 `새싹 땅콩`부터 `차원 수호 땅콩`까지 총 8단계이며, 스킬 레벨 강화와 별도로 누적 적용한다.

### 공통 수치 강화

- 모든 전직 단계에서 스킬 피해가 증가한다.
- 스킬별 피해 증가폭은 기본적으로 단계당 7~10%이며, 사냥·보스·궁극기 전직 구간에서 추가 보너스를 얻는다.
- 전직 단계당 기본 쿨타임이 2% 감소하며 최종 감소율은 최대 22%다.
- 전직 단계당 MP 소모가 1% 감소하며 최종 감소율은 최대 12%다.
- 모든 스킬의 범위 연출 배율은 전직 단계당 12% 증가한다.

### 전투 구조 진화

- 전직할 때마다 스킬별 실제 타격 횟수가 증가한다.
- 사냥 스킬은 실제 최대 타격 대상 수가 증가한다.
- 스킬별 시간차 파동 수가 증가해 공격 전개가 길고 화려해진다.
- 갑각해방의 지속시간과 공격속도 강화량이 증가한다.
- 낙화귀근의 피해 저장률은 32%에서 최대 60%까지 증가한다.
- 황금핵 천단은 전직 3단계와 6단계에서 각각 2단·3단 마무리로 구조가 바뀐다.

### 시각 진화

- 전직마다 검날·꽃잎·꼬투리·검흔 연출 개수가 증가한다.
- 전직 단계마다 스킬 추가 연출의 색상 테마가 변경된다.
- 진화 등급은 `초식 → 개화 → 왕실 → 차원` 순서로 변경된다.
- 전직 성공 순간 현재 전직 이름, 확장 검진, 방사형 검날 연출을 표시한다.
- 스킬 사용 시 현재 진화 등급 문구와 전직 색상의 추가 연출층을 표시한다.

### 상세창 표시

- 현재 타격 수, 최대 대상 수, 범위, 쿨타임, MP를 표시한다.
- 현재 진화 등급과 전직 피해·쿨타임·MP 보너스를 표시한다.
- 다음 전직에서 증가할 타격 수, 대상 수, 파동 수, 연출 개수를 미리 표시한다.
- 상세창의 모든 수치는 실제 전투에서 사용하는 값과 동일해야 한다.

## 구현 규칙

- 보조 땅콩 3명과 겹치는 분신·소환·검령은 사용하지 않는다.
- 사냥 스킬은 다중 대상·전장 제어·화면 전체 공격을 중심으로 한다.
- 보스 스킬은 보스 한 명에게 집중하며 장비의 극저확률 즉사와 겹치는 처형 효과를 넣지 않는다.
- 여덟 스킬은 같은 공용 원형·검격 효과를 재사용하는 수준에 머물지 않고 각자 독립된 연출 순서를 가진다.
- 스킬 레벨 강화와 전직 강화는 중첩 적용한다.
